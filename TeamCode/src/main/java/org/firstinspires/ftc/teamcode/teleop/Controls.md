# Gamepad 1:
* Dpad
    - Up: Slides Up
    - Down: Slides Down
    - Left:
    - Right:
* Left Stick: Drive
* Right Stick: Turn
* A: Toggle Top EndEffector
* B: Toggle Bottom EndEffector
* X: Pivot up
* Y: Pivot down
* Left Bumper: Intake
* Right Bumper: Intake reversed
* Left Trigger: Up and down climb motor
* Right Trigger: Up and down climb servo
